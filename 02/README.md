## 2.7 内存

* 每个内存单元都有一个唯一的编号。
* 单元的编号是这个单元在内存里的位置，通常称为地址（Address）
* 内存单元的编号就是它的地址，习惯上，我们用十六进制标注在它的左侧
* 字节是用来描述二进制序列的长度单位，8比特组成1字节。字节的英文单词是Byte,简写为B

## 2.8 自动计算

* 运算器内部有一个指令指针寄存器，这个寄存器保存着指令的地址。
* 可以根据当前这条指令的地址和长度来计算下一条指令的地址。
* 如果这个数已经在指令中给出了，不需要再次访问内存，那么这个数是立即数
